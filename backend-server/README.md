# HipiPlay Backend / Blockchain

Snapshot seguro del backend blockchain usado por HipiPlay.

## Incluye
- Payment console
- Wallet manager BSC/TRON
- Detector BSC
- Detector TRON/TRC20
- Watcher de acreditacion
- Limpieza unpaid
- Modulo de retiros PAYOUT_HOT en DRY_RUN
- Integracion publica para PWA

## No incluye
- Llaves privadas
- Master key
- Wallets reales cifradas
- Base de datos
- Logs
- Backups
- Tokens de administracion
- Archivos runtime

## Seguridad
Los archivos sensibles deben existir solo en el servidor de produccion y no deben subirse a GitHub.
