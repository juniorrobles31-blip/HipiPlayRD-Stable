# HipiPlay Race Videos

Esta carpeta contiene los videos MP4 usados por el motor visual de carreras.

Requisitos recomendados para cada video:

- Formato: .mp4
- Duración: 30 a 60 segundos
- Vista: carrera hípica real o simulada realista
- Salida desde partidor cerrado
- Caballos saliendo juntos
- Cámara siguiendo la pista ovalada
- Meta visible o entendible
- 5 segundos posteriores a la meta con caballos bajando velocidad
- Evitar que los números reales de los caballos se vean demasiado claros
